== Description ==

= DEV WEB EVENT - A PLUGIN TO CREATE EVENT, VIEW EVENT IN CALENDAR AND ALSO VIEW EVENT AT THE FRONT END.
This plugin using custom post type to create event, using metabox for event date and time meta, custom sortable admin column Event Date and also an admin submenu page under custom post type to view event calendar with event title. For frontend view this plugin using a shortcode.



== Installation & Activation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through Add New Plugin->Upload Plugin->Choose File choose dev-web-event.zip file.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. After successful installation and activation, you will see an Post Type Menu -> Dev Web Events - from this Post Type Menu you can create events with event date and time.
4. Under Dev Web Events post type menu you will see - Event Calendar submenu to view events calendar with event title.
5. After successfully create an event use shortcode - [dev_web_display_events] to any page or post to view all events at the front end.


NOTE: If any 404 error to view event details go to the Settings->Permalinks and click Save Changes button to flush permalink.
